## 📊 五层指标体系

### 第一层：🌊 宏观流动性与货币政策
| 指标 | 监控逻辑 |
|------|----------|
| **DR007** | 银行间资金松紧的核心锚，低于政策利率 = 流动性外溢信号 |
| **10年期国债收益率** | 无风险利率锚，底部回升 = 债→股轮动信号 |
| **央行OMO/MLF净投放** | 监控央行扩表/缩表态度，连续净投放 = 系统性风险归零 |
| **CNH汇率变动率** | 全球风险偏好和外资回流斜率，升值 = 蓝筹估值支撑 |

### 第二层：💰 大盘资金面与风险溢价
| 指标 | 监控逻辑 |
|------|----------|
| **ERP股债风险溢价** | +2σ以上 = 绝对底部区域，中长期配置黄金窗口 |
| **两融余额及买入占比** | <6%情绪冰点 / >11%防范过热踩踏 |
| **北向资金净主动买入** | "聪明钱"左侧建仓的领先指标 |
| **全市场成交额与换手率** | 8000亿/1.5万亿为行情分水岭 |

### 第三层：📊 大盘微观结构与情绪
| 指标 | 监控逻辑 |
|------|----------|
| **期权PCR & IV Skew** | PCR极高+IV回落 = 对冲保护盘买足，现货见底信号 |
| **市场宽度（均线占比）** | 指数跌 + 个股站上均线比例升 = 底背离做多信号 |
| **涨跌停家数比 & 连板晋级率** | 游资情绪晴雨表，冰点 = 指数拐点 |
| **股债相关性轮动指数** | 负相关→正相关修正 = 大类资产配置系统性移位 |

### 第四层：🔄 行业板块轮动
| 指标 | 监控逻辑 |
|------|----------|
| **行业拥挤度** | 单行业成交占比>15% = 右侧必然剧烈回撤 |
| **行业相对强度RS** | 动量排名，识别领涨/领跌板块 |
| **行业ETF净申购** | 机构资金行业配置偏好追踪 |
| **PPI-CPI剪刀差** | 上下游利润分配指南针 |

### 第五层：🎯 个股微观结构
| 指标 | 监控逻辑 |
|------|----------|
| **OFI订单流不平衡** | 主动买卖力量对比，持续正值 = 向上动能 |
| **特质波动率** | 剔除市场beta后的独立波动，极端值预警 |
| **大单占比 & 游资协同** | 成交结构判断主力动向 |
| **移动跟踪止损偏离度** | 动态风险位置量化 |

## 🚀 快速部署

### 1. Fork 或 Clone 本仓库
```bash
git clone https://github.com/gelilong/a-share-quant-monitor.git
cd a-share-quant-monitor
```

### 2. 启用 GitHub Pages
- Settings → Pages → Source: `GitHub Actions` 或 `main` 分支 `/ (root)`
- 等待部署完成

### 3. 启用 GitHub Actions
- Actions 页面 → 启用 workflow
- 数据将在每个交易日自动更新

### 4. 手动运行数据抓取
```bash
pip install -r scripts/requirements.txt
python scripts/fetch_all.py
```
